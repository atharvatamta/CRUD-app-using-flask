# 🗂️ CRUD APP WITH FLASK
---

## 🚀 Features

### 👤 Authentication
- Secure login for admins and users
- Passwords hashed using bcrypt
- Session-based login using Flask-Login


---

## 🧾 Data Fields

Each backup entry includes:

- 📅 **Date of Entry**
- 🙍 **Username**
- 📝 **Details about the Entry**
- 💽 **Number**
